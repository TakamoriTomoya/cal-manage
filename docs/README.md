## 開発の流れ

1. **要件定義書**を確認して、プロジェクトの全体像を把握
2. **Next.js App Router ベストプラクティス**を確認して、開発方針を理解
3. **データ構造定義書**を参照して、型定義を理解
4. **コンポーネント設計書**を参照して、コンポーネントを実装
5. **ローカルストレージ設計書**を参照して、データの保存・取得を実装
6. **開発ガイドライン**に従って、コードを記述
